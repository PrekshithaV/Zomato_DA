# Zomato-Like Food Ordering & Sales Analytics

OVERVIEW :
A full-stack Zomato-like food ordering application with integrated SQL-based sales and customer analytics. Users can place orders through a web interface, while the backend stores and analyzes data to generate business insights.

TECH STACK :
 - Frontend: HTML, CSS
 - Backend: Python (Flask)
 - Database: MySQL
 - Analysis: SQL

KEY FEATURES :
 - Food ordering with cart functionality
 - Order storage in MySQL database
 - Customer and delivery address handling
 - Backend APIs for order processing

ANALYSIS PERFORMED :
 - Total revenue from all orders
 - Revenue by item and customer
 - Revenue by payment method and date
 - Orders by payment status
 - Item-level and customer-level order insights

OUTCOME :
This project demonstrates real-world data analysis using SQL, backend-database integration, and how raw order data can be transformed into actionable business insights.
